# /dd-prep

> 💡 **커넥터 참조:** 이 커맨드는 [`CONNECTORS.md`](../CONNECTORS.md)에 정의된 도구 플레이스홀더를 사용합니다.

## 설명

투자자와의 DD(Due Diligence) 미팅 또는 주요 미팅을 준비합니다. 투자자 배경 조사, 4가지 카테고리별 예상 질문 (시장/재무/팀/딜), 미팅 전략, 데이터룸 체크리스트를 제공합니다.

**핵심 가치:**
- 투자자 thesis & 포트폴리오 기반 맞춤 준비
- 30+ 예상 질문 사전 준비
- 데이터룸 누락 항목 체크
- 미팅 후 후속 조치 플랜

## 사용법

```bash
# 특정 투자자 미팅 준비
/dd-prep "김철수 파트너 (서울투자파트너스)"

# 파트너 미팅
/dd-prep "Sequoia Capital Korea - 파트너 미팅"

# DD 단계
/dd-prep "블루포인트 - DD 시작"

# 다음 미팅 자동
/dd-prep
```

## 필요한 정보

### 자동 수집 (웹 리서치)

- 투자자 배경 & thesis
- 포트폴리오 & 최근 투자
- 파트너 경력 & 관심사

### 사용자 입력 (대화형)

```
1. 미팅 정보
   - 투자자/파트너 이름
   - 미팅 유형 (초기/DD/파트너/최종)
   - 날짜 & 시간

2. 현재 상태
   - 투자 유치 단계
   - 이전 커뮤니케이션 요약
   - 요청받은 자료

3. 우리 현황
   - 최신 트랙션
   - 주요 변경사항
   - 우려 사항
```

1. 트랙션 강조 (최신 업데이트 먼저)

4. 솔직한 챌린지 공유 (신뢰 구축)

**피해야 할 것:**
- 과장된 예측
- 경쟁사 과소평가
- Churn 이슈 회피 (이미 알고 있음)

## 제안 안건 (45분)

**0-5분: 오프닝**
- 참석 감사
- 빠른 업데이트: [대기업 A] POC, MRR 6천만원

**5-15분: DD 후속 논의**
- 제출한 자료에 대한 질문 받기
- 추가 필요 자료 확인

**15-30분: 심화 논의**
- Unit economics 상세 (LTV:CAC, Payback)
- 엔터프라이즈 확장 전략
- Churn 개선 로드맵

**30-40분: 투자 조건 논의**
- 목표 투자 금액: 20억
- 밸류에이션 레인지 논의
- 타임라인 기대치

**40-45분: 다음 단계**
- DD 타임라인 합의
- 레퍼런스 콜 (3명 제안)
- 다음 미팅 일정

## 예상 질문 30개 + 답변 전략

### 카테고리 1: 시장 (8개)

**Q1: TAM/SAM/SOM 계산 근거는?**
답변 전략:
- TAM: $50B (500K US enterprises × $100K avg spend)
- SAM: $10B (50K mid-market × $200K)
- SOM: $1B (Year 3 목표, 200 customers × $5M)
- 근거: Gartner 리포트 + 유사 기업 벤치마크

**Q2: 왜 지금인가?**
답변 전략:
- Remote work → workflow complexity 폭증
- Legacy tools can't handle modern APIs
- 고객 3곳 모두 "pandemic 이후 급박" 언급

**Q3: 시장 성장률은?**
답변 전략:
- 10-15% CAGR (IDC 추정)
- 우리 성장 > 시장 (share 확대 중)

**Q4: 경쟁 심화 리스크는?**
답변 전략:
- 인정: 큰 시장 → 경쟁 불가피
- 방어: First-mover + customer lock-in
- 전환 비용: workflow 데이터 이전 어려움

**Q5: 글로벌 확장 계획?**
답변 전략:
- 현재: US market 검증 (12개월)
- Next: Europe (18개월)
- APAC: 24-36개월

**Q6: [유사기업X]와 비교는?**
답변 전략:
- 유사점: B2B SaaS, mid-market focus
- 차이점: 우리가 20% 빠른 implementation
- 교훈: [유사기업X]의 Enterprise pivot → 우리도 준비 중

**Q7: 고객 세그먼트 우선순위는?**
답변 전략:
- 현재: 200-500명 기업 (Sweet spot)
- 확장: 500-2000명 (Enterprise)
- 이유: CAC vs LTV 최적화

**Q8: 비슷한 회사들의 실패 사례는?**
답변 전략:
- 2010년대: 클라우드 미성숙
- 우리 타이밍: Cloud-native generation
- AI 활용: 당시 불가능했던 automation

### 카테고리 2: 재무 (8개)

**Q9: MRR 성장 지속 가능한가?**
답변 전략:
- 지난 6개월: +25% MoM 일관
- 파이프라인: 다음 3개월 +20% 가시성
- 리스크: CAC 상승 모니터링 중

**Q10: Churn rate 개선 계획은?**
답변 전략:
- 현재: 3% (목표 대비 높음)
- 원인 분석: 초기 PMF 탐색, SMB 실험
- 계획: Enterprise focus → 2% 목표 (6개월)
- 증거: 최근 3개 고객 모두 12개월 계약

**Q11: Unit economics 상세는?**
답변 전략:
- CAC: $300 (Inbound 위주)
- LTV: $15K (ARPU $2.5K × 12개월 / 3% churn × 80% margin)
- LTV:CAC: 5:1 (목표: 3:1 이상)
- Payback: 6개월 (목표: <12개월)

**Q12: Gross Margin이 낮은 이유?**
답변 전략:
- 현재: 78% (SaaS 표준: 75-85%)
- 비용: AWS 15%, CS 7%
- 개선: 규모 경제 → 85% 목표 (Year 2)

**Q13: CAC 상승 리스크는?**
답변 전략:
- 현재: $300 (안정적)
- 채널 mix: 70% Inbound (낮은 CAC)
- 리스크: Paid 확대 시 상승 가능
- 완화: Content marketing 강화

**Q14: 런웨이는?**
답변 전략:
- 현재 자금: 5억원
- 월 Burn: 5천만원
- 런웨이: 10개월
- Series A 타이밍: 6개월 후 시작 (안전)

**Q15: 3년 재무 예측 가정은?**
답변 전략:
- Base 시나리오: +20% MoM → Year 3 ARR 30억
- 가정: 고객 +15%/월, ARPU $2.5K → $4K
- 보수적: Churn 3% 유지 (개선 미반영)

**Q16: Series A 이후 자금 사용처는?**
답변 전략:
- 제품: 40% (엔터프라이즈 기능 3개)
- Sales: 35% (AE 5명, SDR 3명)
- 마케팅: 15% (Content + Partnerships)
- 운영: 10%

## 팁

1. **투자자 thesis를 연구하세요** — 포트폴리오 패턴이 그들의 관심사를 보여줍니다.

2. **솔직한 챌린지 공유가 신뢰를 만듭니다** — Churn 이슈를 숨기지 말고 개선 계획을 보여주세요.

3. **숫자를 암기하세요** — CAC, LTV, Churn 등 핵심 메트릭스는 즉답할 수 있어야 합니다.

4. **시나리오를 준비하세요** — "만약 X하면?"질문에 대비한 대안을 준비하세요.

5. **팀 미팅 연습** — 동료와 모의 Q&A로 답변 다듬기.

6. **데이터룸을 미리 준비하세요** — 요청 즉시 제공할 수 있어야 프로페셔널합니다.

7. **레퍼런스를 사전 조율하세요** — 고객에게 미리 양해 구하고 포지티브 피드백 확보.

8. **후속 조치는 24시간 이내** — 감사 이메일 + 요청 자료는 즉시.

9. **다른 파트너 만나기** — DD 단계에서 펀드 내 여러 관계자 확보 (단일 스레드 회피).

10. **타임라인 관리** — DD는 2-3주가 적절. 너무 길면 관심 저하, 너무 짧으면 졸속.

## 에이전트 병렬 실행

이 커맨드는 실행 시 다음 에이전트를 병렬로 호출합니다:

| 에이전트 | 역할 | 출력 |
|---------|------|------|
| `investor-researcher` | 투자자 최신 리서치 — thesis, 포트폴리오, 최근 투자, 개인화 훅 | 투자자 심층 프로필 |
| `vc-question-prepper` | 시장/재무/팀/딜 4개 카테고리 × 30개 예상 질문 + 답변 전략 | Q&A 준비 자료 |

**실행 구조:**

```
[/dd-prep "김철수 파트너"] 실행
         │
         ├── investor-researcher ─── 웹 리서치 → 투자자 프로필 + 개인화 훅
         │         ↓ 포트폴리오 패턴, 최근 관심사
         └── vc-question-prepper ─── 30개 예상 질문 + 답변 전략
                   ↓
         [미팅 준비 통합 보고서]
```

**특정 VC 특화:**

```bash
/dd-prep "알토스벤처스 파트너 미팅"
# → investor-researcher: 알토스 포트폴리오 분석 + 파트너 관심사 파악
# → vc-question-prepper: 알토스 투자 철학 기반 예상 질문 특화 생성
```

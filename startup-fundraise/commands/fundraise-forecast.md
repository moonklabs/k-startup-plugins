# /fundraise-forecast

> 💡 **커넥터 참조:** 이 커맨드는 [`CONNECTORS.md`](../CONNECTORS.md)에 정의된 도구 플레이스홀더를 사용합니다.

## 설명

투자 유치 예측을 생성합니다. 단계별 확률을 적용한 가중 예측, 3가지 시나리오(최선/예상/최악), 런웨이 교차점 분석, 목표 대비 갭 분석 및 권고 사항을 제공합니다.

**핵심 가치:**
- 객관적 확률 기반 예측 (희망 vs 현실)
- 런웨이 리스크 조기 경보
- 시나리오별 대응 전략
- 투자자별 성사 가능성 평가

## 사용법

```bash
# 현재 라운드 예측
/fundraise-forecast

# 특정 라운드
/fundraise-forecast "Seed 라운드"

# 기간 지정
/fundraise-forecast "Q1 2025"

# 시나리오 비교
/fundraise-forecast "시나리오 분석"
```

## 필요한 정보

### 1단계: 파이프라인 데이터

**옵션 A: CSV 업로드**
```csv
투자자명,VC펀드,단계,예상금액,마감예상일,Thesis적합도,메모

이영희,블루포인트,DD,10억,2024-12-31,HIGH,DD 진행 중
...
```

**옵션 B: 투자자 리스트 붙여넣기**
```
김철수 (서울투자파트너스) - 텀시트 - 5억 - HIGH
이영희 (블루포인트) - DD - 10억 - HIGH
박지훈 (프라이머) - 미팅 - 3억 - MEDIUM
...
```

**옵션 C: 파이프라인 설명**
```
"Seed 20억 모집 중입니다. 15명 파이프라인 중 2명은 텀시트 단계 (15억),
3명은 DD (20억), 5명은 미팅 (25억), 나머지는 초기 단계입니다."
```

### 2단계: 목표 & 현황

```
- 목표 자금: 20억원
- 라운드: Seed
- 목표 마감일: 2025-01-31
- 현재 확약 (Soft Commit): 5억원
- 현재 런웨이: 10개월
- 월 Burn rate: 5천만원
```

## 팁

1. **보수적으로 예측하세요** — 희망 대신 확률 기반. 실제는 보통 예상보다 낮습니다.

2. **3x 커버리지 유지** — 목표의 3배 파이프라인이 안전. 일부는 항상 이탈합니다.

3. **주간 업데이트** — 파이프라인은 빠르게 변합니다. 매주 예측 업데이트.

4. **런웨이 = 데드라인** — 런웨이 3개월 전에 라운드 시작, 클로징 완료 목표.

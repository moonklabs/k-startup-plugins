# /fundraise-pipeline

> 💡 **커넥터 참조:** 이 커맨드는 [`CONNECTORS.md`](../CONNECTORS.md)에 정의된 도구 플레이스홀더를 사용합니다. 연결된 도구가 많을수록 더 강력해지지만, 수동 입력만으로도 효과적으로 작동합니다.

## 설명

투자자 파이프라인의 건강 상태를 4차원(진행속도, 응답률, Thesis 적합도, 관계깊이)으로 평가하고, 3x 커버리지 규칙을 검증하며, 집중해야 할 투자자와 리스크 영역을 식별합니다.

투자유치 운영 단계별 사용법과 실사용 PASS/FAIL 기준은 [Founder Fundraising Operating Use Cases](../skills/fundraising-process/references/founder-fundraising-operating-use-cases.md)를 기준으로 합니다.

**핵심 가치:**
- 건강점수 100점 체계로 파이프라인 상태 정량화
- 3x 커버리지 규칙 검증 (목표 대비 파이프라인)
- 정체/무응답 투자자 조기 플래그
- 단계별 병목 지점 식별

## 사용법

```bash
# 전체 파이프라인 리뷰
/fundraise-pipeline

# 특정 단계만
/fundraise-pipeline "미팅"

# 특정 라운드
/fundraise-pipeline "Seed 라운드"

# 리스크 투자자만
/fundraise-pipeline "리스크"
```

## 건강점수 계산 방식

### 1. 진행속도 (25점)

```
기준:
- 각 단계별 평균 체류 일수 목표:
  - 발굴 → 접촉: 7-14일
  - 접촉 → 미팅: 14-21일
  - 미팅 → DD: 21-30일
  - DD → 텀시트: 30-45일
  - 텀시트 → 클로징: 7-14일

감점:
- 목표 초과 시: -1점/투자자 (최대 -10점)
- 30일+ 정체: -2점/투자자 (최대 -15점)

예시:
- 15명 중 3명이 30일+ 정체 → -6점
- 5명이 목표 초과 → -5점
- 점수: 25 - 11 = 14점
```

### 2. 응답률 (25점)

```
기준:
- 응답률 = (응답한 투자자 / 접촉한 투자자) × 100%
- 목표: 75% 이상

점수:
- 90%+: 25점
- 80-90%: 20점
- 75-80%: 18점
- 70-75%: 15점
- 60-70%: 10점
- <60%: 5점

예시:
- 15명 중 12명 응답 = 80% → 20점
```

### 3. Thesis 적합도 (25점)

```
기준:
- HIGH 적합도 비율 목표: 60% 이상
- MEDIUM: 30-40%
- LOW: <10%

점수:
- HIGH 60%+ & LOW <10%: 25점
- HIGH 50-60% & LOW <15%: 20점
- HIGH 40-50%: 15점
- HIGH <40%: 10점
- LOW >20%: -5점

예시:
- HIGH 8명(53%), MEDIUM 5명(33%), LOW 2명(13%)
- 점수: 20점
```

### 4. 관계깊이 (25점)

```
기준:
- 웜 인트로 비율 목표: 50% 이상
- 다중 스레드 비율 목표: 70% 이상 (DD 단계 이상)

점수:
- 웜 인트로 50%+: +12점
- 웜 인트로 30-50%: +8점
- 웜 인트로 <30%: +4점

- 다중 스레드 70%+: +13점
- 다중 스레드 50-70%: +8점
- 다중 스레드 <50%: +4점

예시:
- 웜 인트로 6명(40%) → +8점
- 다중 스레드 2명/5명(40% - DD 이상만 계산) → +4점
- 총점: 12점
```

## 3x 커버리지 규칙

### 원칙

```
필요 파이프라인 = 목표 자금 × 3배

이유:
- 평균 클로징률: 30-40%
- 리스크 버퍼: 일부 투자자 이탈, 금액 축소
```

### 적합도별 가중치

```
HIGH 적합도: 1.0x (그대로)
MEDIUM 적합도: 0.5x (절반만 계산)
LOW 적합도: 0.1x (거의 무시)

예시:
- HIGH: 45억 × 1.0 = 45억
- MEDIUM: 25억 × 0.5 = 12.5억
- LOW: 5억 × 0.1 = 0.5억
- 가중 파이프라인: 58억
- 커버리지: 58억 / 20억 = 2.9x (부족)

→ HIGH 적합도 투자자 추가 필요
```

## 팁

1. **주간 리뷰 필수** — 파이프라인은 빠르게 변합니다. 매주 월요일 리뷰로 조기 경보.

2. **3x 커버리지 유지** — 일부 클로징하면 즉시 신규 타겟 추가하여 커버리지 유지.

3. **LOW 적합도는 과감히 제거** — 시간과 에너지는 유한합니다. HIGH에 집중.

4. **웜 인트로가 압도적 우위** — 콜드 이메일 10건 < 웜 인트로 1건.

5. **정체 = 관심 없음** — 30일+ 정체는 사실상 거절입니다. 제거하고 다음으로.

6. **다중 스레드 필수** — DD 단계 이상은 반드시 펀드 내 여러 관계자 확보.

7. **단계별 전환율 추적** — 병목 지점을 찾아 집중 개선.

8. **파이프라인 건강 = 투자 유치 속도** — 건강점수 80+ 유지 시 목표 달성 확률 높음.

9. **응답률은 Thesis 적합도의 선행지표** — 응답률 낮으면 적합도 재점검.

10. **매달 신규 타겟 추가** — 파이프라인은 살아있는 생명체. 지속적 보충 필요.

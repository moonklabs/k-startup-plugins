# /lead-dashboard

> 익숙하지 않은 플레이스홀더가 보이거나 연결된 도구를 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 참조하세요.

투자자 리드 파이프라인 전체 현황을 분석하고, 건강점수를 계산하며, 집중해야 할 영역에 대한 실행 가능한 권고 사항을 제공합니다.

> **`/fundraise-pipeline`과의 구분:** 이 커맨드는 매일/수시로 보는 **빠른 스냅샷 대시보드**입니다. 4차원 건강점수, 3x 커버리지 검증, 병목 심층 분석이 필요하면 `/fundraise-pipeline`을 사용하세요.

## 사용법

```
/lead-dashboard
/lead-dashboard 미팅 단계
/lead-dashboard 정체
```

전체 또는 특정 필터(단계, 상태)를 지정하세요.

```
├─────────────────────────────────────────────────────────────────┤
│  강화 모드 (도구 연결 시)                                         │
│  + ~~CRM: 투자자 리드 자동 가져오기, 활동 이력 추적               │

│  + ~~calendar: 미팅 일정, 다음 접촉 일자 자동 업데이트             │
└─────────────────────────────────────────────────────────────────┘
```

## 필요한 정보

**옵션 A: ~~CRM 연결 시**
자동으로 투자자 리드 데이터를 가져옵니다.

**옵션 B: 스프레드시트 업로드**
CSV 또는 스프레드시트를 업로드하세요. 유용한 필드:
- 투자자명 (VC 펀드/파트너)
- 단계 (발굴/접촉/미팅/심사DD/텀시트/클로징/패스)
- 최근 활동일
- 다음 액션
- Thesis 적합도 (HIGH/MEDIUM/LOW)
- 접근 경로 (웜인트로/콜드)

**옵션 C: 리드 붙여넣기**
```
Sequoia Capital — 미팅 단계 — 최근 활동 1/15 — 다음: DD 자료 전달
Andreessen Horowitz — 접촉 단계 — 최근 활동 1/10 — 다음: 파트너 미팅 요청
Y Combinator — 심사DD — 최근 활동 1/20 — 다음: 파트너 컨센서스 대기
```

## 건강점수 산정 방식

### 1. 진행 속도 (25점)

```
- 10점: 동일 단계 30일+ 정체 투자자 비율
- 15점: 평균 단계 체류 기간
```

### 2. 응답률 (25점)

```
- 15점: 14일+ 무응답 투자자 비율
- 10점: 평균 응답 시간
```

### 3. Thesis 적합도 (25점)

```
- HIGH 비율 ≥ 70% → 25점
- HIGH 비율 50-70% → 15점
- HIGH 비율 < 50% → 5점
```

### 4. 관계 깊이 (25점)

```
- 다중 파트너 접촉 비율
- 포트폴리오/기존 투자자 연결 비율
```

## ~~CRM 연결 시

- 투자자 리드를 자동으로 가져옵니다
- 활동 이력과 커뮤니케이션 기록을 자동 추적합니다
- 단계 전환, 다음 액션 업데이트를 제안합니다
- 시간 경과에 따른 전환율 추이를 분석합니다

## 팁

1. **주간 대시보드 리뷰** — 매주 월요일 `/lead-dashboard` 실행으로 주간 계획 수립
2. **커버리지 3x 유지** — 타겟 마감액의 3배 파이프라인 유지 (전환율 1-3% 고려)
3. **정체 투자자 정리** — 30일+ 무활동 투자자는 재접촉 또는 패스 처리
4. **전환율 벤치마크 추적** — 자신의 전환율 패턴 파악하여 딜소싱 규모 조정
5. **패스 사유 분석** — 패턴 파악하여 타게팅 정확도 개선

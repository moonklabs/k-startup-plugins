# /find-vc

> 익숙하지 않은 플레이스홀더가 보이거나 연결된 도구를 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 참조하세요.

창업자-facing 투자자 탐색 커맨드입니다. 내부적으로 `/deal-sourcing`의 리서치 방식을 따르되, 사용자는 “딜소싱”이라는 업계 용어를 몰라도 됩니다.

목표는 단순 VC 리스트가 아니라, **누구를 어떤 순서로 만나야 VC funding 성공 확률이 올라가는지**를 정하는 것입니다.

## 사용법

```bash
/find-vc "뭉클랩 AI B2B SaaS Seed Pre-A Korea"
/find-vc "TIPS 가능 AC/VC 찾기"
/find-vc "runway 5개월, 20억 Seed 라운드 리드 후보"
```

## 작동 방식

```
┌─────────────────────────────────────────────────────────────────┐
│                          FIND VC                                 │
├─────────────────────────────────────────────────────────────────┤
│  단독 사용 (웹 검색으로 항상 작동)                                 │
│  ✓ VCS 투자자 검색 + 모태출자펀드 운용사 찾기로 후보군 생성        │
│  ✓ KVIC FundFinder, KVCA DIVA, TIPS 공식 사이트 교차 대사          │
│  ✓ 단계/섹터/TIPS/펀드 근거/접근 경로/미팅 순서 6축 평가           │
│  ✓ PRACTICE → VALIDATE → CLOSE 순서로 미팅 리스트 정렬            │
├─────────────────────────────────────────────────────────────────┤
│  강화 모드 (도구 연결 시)                                          │
│  + ~~fund disclosure: 로컬 공시 DB에서 펀드/TIPS 근거 즉시 조회    │
│  + ~~CRM: 기존 접촉 이력, 웜인트로 경로 자동 매핑                  │
│  + ~~knowledge base: 팀 문서에서 연결자/인트로 이력 검색           │
└─────────────────────────────────────────────────────────────────┘
```

## 한국형 확인 축

| 축 | 질문 | 근거 |
|---|---|---|
| 단계 적합도 | Seed/Pre-A에 실제로 투자하는가? | VCS 업력/창업초기 조건, 최근 투자, 포트폴리오 |
| 섹터 적합도 | 우리 wedge와 펀드 목적이 맞는가? | VCS 주요투자업종/투자성격, thesis, 조합명 |
| TIPS 가능성 | 운영사이며 추천 가능성이 있는가? | TIPS Partners, 운영사 접수처, 최근 선정팀 |
| 펀드 근거 | 실제 투자 가능한 펀드/조합 근거가 있는가? | VCS 모태출자펀드, KVIC, KVCA DIVA, 공식 공시 |
| 접근 경로 | 콜드보다 강한 소개 경로가 있는가? | 기존 투자자, 고객, 포트폴리오 founder |
| 미팅 순서 | 바로 리드 후보를 만날지, 연습이 필요한가? | objection readiness, IR 완성도 |

## 미팅 단계

| 단계 | 목적 | 예시 |
|---|---|---|
| `PRACTICE` | 반박 수집, 메시지 검증 | 최우선은 아니지만 적합한 VC/AC |
| `VALIDATE` | 투자 가능성, TIPS/펀드 목적, thesis 검증 | 우리 분야/단계가 맞는 후보 |
| `CLOSE` | 리드 투자자 확보 | 정말 받고 싶은 VC/AC |

## 실행 규칙

- 한국 투자자 후보 발견은 [VCS 투자자 검색](https://www.vcs.go.kr/web/portal/investor/list)을 먼저 사용합니다. 회사 정보를 주요투자업종, 투자성격, 업력, 지역, 투자자 구분으로 변환해 검색합니다.
- 모태펀드가 관련되면 [VCS 모태출자펀드 운용사 찾기](https://www.vcs.go.kr/web/portal/rsh/list)에서 펀드명, 결성일, 결성총액, 투자금액, 존속기간, 운용사를 확인합니다.
- VCS 후보를 KVIC FundFinder, KVCA DIVA, TIPS 공식 사이트, 투자사 공식 홈페이지와 대사합니다. 한 소스의 결과만으로 현재 투자 가능성이나 잔여 투자 여력을 확정하지 않습니다.
- 공식 근거와 투자 가능성을 분리합니다. VCS/KVIC/KVCA 근거는 “볼 가치”의 증거이지 “투자 확정”이 아닙니다.
- TIPS는 투자자/운영사 관점에서 봅니다. 운영사의 투자심사, 투자 확약, 추천 가능성, R&D 과제 적합성을 함께 확인합니다.
- 이름 있는 VC를 앞에 두지 말고, 연습/검증/클로징 순서로 정렬합니다.
- 근거가 없으면 `needs_evidence`와 필요한 import/search 액션을 명시합니다.
- VCS 응답에 포함될 수 있는 대표자명, 전화번호, 이메일, 상세주소는 수집하거나 출력하지 않습니다. 투자사/펀드 식별자, 명칭, 유형, 업종, 규모, 결성일, 투자금액, 존속기간, 지역, 공식 홈페이지까지만 사용합니다.
- VCS의 페이지 내부 JSON 검색 경로는 공식 공개 API 계약이 아닙니다. 브라우저 검색 또는 사용자 요청 단위의 저빈도 조회만 사용하고, 대량 순회·재배포용 미러 DB를 만들지 않습니다.

## 수집 순서

1. 회사의 설립연차, 업종, 투자 단계, 지역, 목표 금액을 정규화합니다.
   - 설립일을 모르면 업력 구간을 추측하지 않고 필터를 생략한 뒤 `company_age_missing`으로 표시합니다.
2. VCS 투자자 검색으로 넓은 후보군을 만듭니다. Seed/Pre-A는 `창업초기`와 실제 회사 업력 구간을 우선 적용하고, AI 기업은 `ICT서비스`와 `AI·빅데이터`를 함께 봅니다.
3. VCS 모태출자펀드 검색으로 후보 운용사의 펀드 근거를 연결합니다.
4. KVIC FundFinder로 정책 목적/세부 출자 분야를 확인하고 KVCA DIVA로 특정 운용사의 조합 공시를 대사합니다.
5. 최근 투자, 공식 홈페이지, TIPS 운영사 여부, 웜인트로 경로를 보강한 뒤 `PRACTICE`, `VALIDATE`, `CLOSE`로 정렬합니다.

VCS의 정확한 검색 변수와 필드 매핑은 `deal-sourcing` 스킬의 [VCS 검색 파라미터 카탈로그](../skills/deal-sourcing/references/vcs-search-parameter-catalog.md)를 따릅니다.

# MoonkLabs Seed/Pre-A Fundraising Playbook

이 문서는 딜소싱 스킬의 회귀 샘플이다. MoonkLabs처럼 한국 기반 AI/MCP/API wrapper/운영 데이터 자동화 스타트업이 Seed-Pre-A 투자 유치를 준비할 때, KVIC FundFinder와 KVCA DIVA를 어떻게 함께 써서 후보 VC를 찾고 미팅 순서를 정하는지 보여준다.

## 전제

- 회사: MoonkLabs / 뭉클랩
- 단계: Seed-Pre-A 준비
- 포지셔닝: 단순 API wrapper나 SI가 아니라, 한국 SMB와 스타트업의 결제, 정산, 광고, 분석, 세무, 문서 데이터를 AI agent가 사용할 수 있게 만드는 운영 데이터 계층
- 초기 wedge: Toss Payments, PortOne, Inicis 등 PG 매출/정산/결제상세 조회와 광고/analytics/CRM 데이터 집계
- 검증 필요: 실제 매출, 유료 고객, 사용 빈도, 데이터 연결 수, 고객별 반복 업무 절감 시간

## 공식 데이터 조회 흐름

1. 회사 프로필을 FundFinder 조건 코드로 변환한다.
2. KVIC FundFinder에서 조건별 펀드 후보를 조회한다.
3. 후보 운용사와 펀드명을 KVCA DIVA 조합현황에서 대사한다.
4. 결성일, 만기일, 결성약정총액, 투자 집행액, thesis 적합도, 접근 경로를 함께 점수화한다.
5. KVCA에 없는 펀드는 존재하지 않는다고 단정하지 않고, 별도 펀드 형태인지 확인 대상으로 둔다.

## 미팅 순서 전략

### 1단계: 연습과 메시지 검증

대상: 무신사파트너스, 롯데벤처스, 일부 클레어보이언트벤처스

목표:
- "API wrapper인가, SI인가, venture-scale product인가" 반박을 먼저 수집한다.
- commerce/SMB 운영 데이터 문제를 투자자가 이해하는 언어로 바꾼다.
- 데모에서 결제/정산/광고/CRM 데이터가 한 화면으로 연결되는 순간을 검증한다.

성공 기준:
- 투자자가 고객 pain과 반복 사용 이유를 자기 말로 되풀이한다.
- 다음 미팅 또는 관련 포트폴리오 소개 요청을 받는다.

### 2단계: 투자 가능성 검증

대상: 수인베스트먼트캐피탈, 클레어보이언트벤처스, 스파크랩 일부 파트너

목표:
- check size, lead/follow 가능성, 다음 라운드까지의 milestone을 확인한다.
- 데이터 연결 수, 반복 사용률, 고객별 업무 절감 시간 같은 KPI가 투자 판단에 충분한지 확인한다.
- 기술사업화/IP narrative가 필요한 VC에는 connector catalog, parsing pipeline, evidence store 구조를 보여준다.

성공 기준:
- DD 요청 자료 목록을 받는다.
- 파트너 미팅 또는 IC 전 사전 검토로 넘어간다.

### 3단계: 리드 후보 클로징

대상: 티인베스트먼트, 스파크랩, 수인베스트먼트캐피탈

목표:
- 목표 조달 금액, 리드 조건, follow 투자자 구성, SAFE/RCPS 선호를 맞춘다.
- AI/클라우드/운영 데이터 thesis에 맞춰 MoonkLabs를 "Korean business data layer for AI agents"로 선명하게 포지셔닝한다.
- 초기 고객 증거와 제품 데모를 같은 흐름으로 보여준다.

성공 기준:
- term sheet 논의 또는 구체적인 조건 협상으로 진입한다.
- follow 투자자 소개를 받는다.

## 미팅 준비 체크리스트

- 12장 이하 deck: 문제, 고객, 데모, traction, 시장, 경쟁, business model, go-to-market, team, ask
- 3분 데모: PG 매출/정산/결제상세 조회에서 시작해 광고/analytics/CRM 데이터까지 연결
- VC별 1페이지 메모: 해당 펀드명, 결성일, 결성총액, thesis fit, 왜 지금 만나야 하는지
- KPI: 연결된 서비스 수, 고객 수, 유료 전환, 사용 빈도, 자동화된 리포트 수, 절감 시간
- 방어 논리: "wrapper는 commoditized" 반박에 대한 workflow, data model, local Korean API coverage, evidence store, MCP/skill distribution 답변
- 요청: 투자 금액, runway, 이번 라운드 milestone, lead/follow 선호

## 미팅 후속 조치

당일:
- 감사 메일
- 투자자가 물은 핵심 질문 3개
- 요청 받은 자료
- 다음 미팅 제안 일자

48시간 이내:
- VC별 objection log 업데이트
- deck/FAQ 수정
- 다음 VC 미팅에서 검증할 가설 1-2개 선정

주간:
- Stage 1 objection을 Stage 2/3 pitch에 반영
- DIVA/FundFinder 근거가 바뀐 후보는 snapshot 날짜와 함께 갱신
- 리드 후보에게는 경쟁 관심도와 일정감을 투명하게 공유하되 과장하지 않음

## 스킬 출력 검증 기준

딜소싱 스킬이 이 샘플을 제대로 처리했다면 결과에는 다음이 포함되어야 한다.

- FundFinder 조건 코드와 코드 선택 이유
- KVIC 후보 펀드 목록과 운용사
- KVCA DIVA 대사 여부와 `ASCT_ID`
- KVCA 미확인 후보에 대한 caveat
- 1단계, 2단계, 3단계 미팅 순서
- VC별 맞춤 positioning과 준비 자료
- 후속 조치 체크리스트

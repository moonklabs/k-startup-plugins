# VCS 검색 파라미터 카탈로그

검증일: 2026-07-10

벤처투자종합포털(VCS)은 한국 투자자 후보를 업종, 투자성격, 회사 업력, 지역, 투자자 유형으로 찾고 모태출자펀드 운용사를 함께 확인할 수 있는 공식 포털이다.

- 투자자 검색: https://www.vcs.go.kr/web/portal/investor/list
- 모태출자펀드 운용사 찾기: https://www.vcs.go.kr/web/portal/rsh/list

이 카탈로그의 `/search` 경로는 페이지가 사용하는 내부 JSON 요청이다. 문서화된 공개 API 계약이 아니므로 변경될 수 있다. 브라우저 검색을 기본으로 하고, 자동화는 사용자 요청 단위의 저빈도 조회 또는 사용자가 저장한 snapshot import로 제한한다.

## 스타트업 입력 매핑

| 사용자 입력 | VCS 조건 | 적용 규칙 |
|---|---|---|
| 사업 분야 | `comIndCdArr` | 가장 가까운 주요투자업종 1~2개를 선택한다. AI B2B SaaS는 `2`(ICT서비스)를 기본으로 본다. |
| 투자 목적/정책 분야 | `invstCharcCdArr` | AI는 `28`, 창업초기는 `20`, ICT는 `06`처럼 복수 선택할 수 있다. |
| 설립일 | `comBzcarrCdArr` | 현재 날짜 기준 회사 업력을 계산한 뒤 정확한 구간 하나를 선택한다. 설립일을 모르면 추측하지 않고 생략한다. |
| 투자사 소재지 | `localCdArr` | 지역 펀드나 대면 접근성이 중요할 때만 제한한다. |
| 투자자 유형 | `invstorTpCdArr` | VC, 신기술사, LLC, 해외VC, 기타운용사를 구분한다. AC 여부는 결과의 `strtplanerYn`도 확인한다. |
| 목표 펀드 규모 | `operScaleCdArr` | 투자사 운영규모 또는 모태펀드 결성총액 구간으로 사용한다. 체크사이즈와 동일한 값으로 해석하지 않는다. |

## 투자자 검색

페이지는 `GET /web/portal/investor/search`를 사용한다.

| 변수 | 값 | 의미 |
|---|---|---|
| `tabMenu` | `1` / `2` | 투자사 보기 / 펀드·조합 보기 |
| `cp` | 양의 정수 | 페이지 번호. 기본 page size는 10이다. |
| `comIndCdArr` | `1`~`9` | ICT제조, ICT서비스, 전기·기계·장비, 화학·소재, 바이오·의료, 영상·공연·음반, 게임, 유통·서비스, 기타 |
| `invstComLocCdArr` | `1`~`3` | 수도권, 5대 광역시, 그 외 지방 |
| `localCdArr` | 지역 코드 | 서울 `01`, 부산 `02`, 대구 `03`, 인천 `04`, 광주 `05`, 대전 `06`, 울산 `07`, 강원 `08`, 경기 `09`, 경남 `10`, 경북 `11`, 전남 `12`, 전북 `13`, 제주 `14`, 충남 `15`, 충북 `16`, 세종 `17`, 기타·해외 `99` |
| `operScaleCdArr` | `1`~`5` | 100억원 미만, 100~300억원, 300~500억원, 500~1,000억원, 1,000억원 이상 |
| `comBzcarrCdArr` | `1`~`6` | 1년 이하, 1~3년, 3~5년, 5~7년, 7~14년, 14년 초과 |
| `invstorTpCdArr` | `1`,`2`,`3`,`4`,`5`,`6`,`7`,`9` | 벤처투자회사, 신기술사, CRC, LLC, 해외VC, 개인, 기타법인, 기타운용사 |
| `sc` | `operInstNm` / `fundNm` | 명칭검색 대상 |
| `sv` | 문자열 | 투자사 또는 펀드명 |

## 모태출자펀드 운용사 검색

| 결과 필드 | 저장/표시 | 의미 |

| `fundNm` | 허용 | 펀드명 |

## Seed/Pre-A 기본 검색 예시

AI B2B SaaS 초기기업은 다음 조합에서 시작한다.

1. 투자자 검색: `comIndCdArr=2`, `invstCharcCdArr=28`, `invstCharcCdArr=20`, 실제 회사 업력에 맞는 `comBzcarrCdArr`.
2. 결과가 너무 많으면 투자자 유형과 지역을 추가한다. 운영규모만으로 초기투자 적합도를 단정하지 않는다.
3. 모태출자펀드 검색: `comIndCdArr=2`로 후보 펀드와 운용사를 찾는다.
4. 오래된 펀드는 `결성일 + 존속기간`을 계산해 현재 운용 가능성을 확인한다.
5. `결성총액 - 투자금액`은 단순 참고치다. 관리보수, 후속투자 reserve, 미집행 약정, 평가시점 차이가 있으므로 dry powder로 확정하지 않는다.

## 수집 및 증거 규칙

- 화면 URL, 선택한 코드/라벨, 조회시각, 결과 식별자를 함께 기록한다.
- VCS 결과는 공식 포털 화면 근거이지만 현재 투자 의사, 신규투자 가능 금액, TIPS 추천 가능성을 보장하지 않는다.
- KVIC FundFinder, KVCA DIVA, TIPS 공식 사이트, 투자사 공식 홈페이지로 대사한다.
- VCS `robots.txt`는 포털 경로를 허용하지만 이용약관과 저작권 방침은 무단 복제·제공을 제한한다. 사용자 요청 단위 조회와 개인 로컬 snapshot은 허용 경계로 보고, 대량 미러링·재배포·상업 데이터셋 제작은 사전 허가 없이는 하지 않는다.
- 내부 JSON 경로가 실패하거나 응답 형식이 달라지면 브라우저 UI로 전환하고 `source_contract_changed`를 기록한다.

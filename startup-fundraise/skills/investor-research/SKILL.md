# 투자자 리서치 (Investor Research)

아웃리치 전에 VC 펀드 또는 파트너에 대한 완전한 그림을 제공합니다. 이 스킬은 웹 검색만으로도 작동하며, 한국 VC는 공식 펀드 근거와 KVCA 조합 대사 결과를 웹 리서치와 분리해 표시합니다.

## 작동 방식

```
┌─────────────────────────────────────────────────────────────────┐
│                   INVESTOR RESEARCH                              │
├─────────────────────────────────────────────────────────────────┤
│  기본 기능 (웹 검색으로 단독 작동)                                 │
│  ✓ 펀드 개요: Thesis, 펀드 규모, 투자 단계                       │
│  ✓ 포트폴리오 분석: 대표 기업, 섹터 패턴                          │
│  ✓ 주요 파트너: 배경, 투자 이력, LinkedIn                        │
│  ✓ 최신 뉴스: 신규 펀드, 최근 투자, 발표                          │
│  ✓ 한국 VC 공식 근거: KVIC 조건, KVCA ASCT_ID, 조합 만기          │
│  ✓ Thesis 적합도: 4차원 평가 (섹터·단계·체크·지역)               │
├─────────────────────────────────────────────────────────────────┤
│  강화 모드 (도구 연결 시)                                         │
│  + ~~data enrichment: 정확한 펀드 데이터, 전체 포트폴리오, 투자 이력│
│  + ~~CRM: 기존 인트로 경로, 공동 포트폴리오 자동 매핑              │
└─────────────────────────────────────────────────────────────────┘
```

## 커넥터 (선택사항)

도구를 연결하여 이 스킬을 강화하세요:

| 커넥터 | 추가 기능 |
|--------|----------|
| **데이터 보강** | THE VC (투자 라운드·포트폴리오), 혁신의숲 (성장 지표), OpenDART (상장사 공시) — 웹 검색 기반 접근, OpenDART는 MCP 연결 가능 |

> **커넥터가 없나요?** 문제없습니다. 웹 검색만으로도 모든 VC에 대한 충실한 리서치를 제공합니다.

```
## 경쟁사 투자 체크

이 VC가 우리 경쟁사에 투자했는지 확인:

| 경쟁사 | 투자 여부 | 투자 단계 | 시사점 |
|--------|----------|----------|--------|
| [경쟁사 A] | ✅ Yes | Series A | ⚠️ 동일 카테고리 재투자 unlikely |
| [경쟁사 B] | ❌ No | — | ✅ 기회 열림 |

**권장:**
- 경쟁사 투자 있으면 → 파트너십 각도, 또는 제외
- 없으면 → "아직 이 카테고리에 투자 안 함" 각도

### 각도 5: 공식 펀드 근거와 Why Now

한국 VC에서는 KVIC/KVCA 근거를 과장 없이 개인화 훅으로 사용합니다.

```
[VC명]이 보유한 [펀드명]은 [초기기업/AI/기술사업화] 조건과 맞고,
[결성일/만기/결성총액] 기준으로 지금 [회사명]의 [라운드]와 대화할 이유가 있습니다.
```

공식 근거는 "투자 확정 가능성"이 아니라 "대화 우선순위와 thesis 적합성"의 근거로만 사용합니다.
```

## 실행 흐름

### 1단계: 요청 분석

```
조사 대상 파악:
- "Sequoia Capital 리서치" → 펀드 전체
- "Marc Andreessen at a16z" → 특정 파트너
- "fintech seed investors in Korea" → 카테고리 (deal-sourcing으로 라우팅)
```

### 2단계: 웹 검색 (항상 실행)

```
다음 검색 병렬 실행:
1. "[VC 펀드명]" → 공식 웹사이트, About 페이지
2. "[VC 펀드명] thesis investment strategy"
3. "[VC 펀드명] portfolio companies"
4. "[VC 펀드명] recent investments 2024 2025"
5. "[VC 펀드명] partners team"
6. "[파트너명] [VC] LinkedIn"
7. "[VC 펀드명] fund size AUM"
8. "[VC 펀드명] news announcements"
```

**추출 정보:**
- Thesis, 투자 초점
- 포트폴리오 대표 기업 (5-10개)
- 주요 파트너 (이름, 직함, LinkedIn)
- 펀드 규모, 최근 펀드 조성
- 최근 투자 발표 (최근 90일)
- 전형적 체크사이즈, 투자 단계

```
1. KVIC FundFinder
   - http://fundfinder.k-vic.co.kr/rsh/rsh/RshMacFnd
   - 회사의 설립연차/사업 분야에 맞는 모태펀드 출자 펀드 확인
   - `ASCT_CLSS_GRP_CD_FND`, `ASCT_CLSS_CD_FND` 조건 코드를 기록
   - 펀드명, 운용사, 주력 투자 분야, 결성일, 결성총액, 투자 집행액 추출

2. KVCA DIVA 항목별공시
   - http://diva.kvca.or.kr/div/cmn/DivDisclsMainInq
   - 항목별공시 → 조합현황 → VC명 검색
   - VC별 보유 벤처투자조합 현황 확인
```

**리포트 반영:**
- `공식 펀드 근거`: KVIC/KVCA 확인 여부
- `FundFinder 조회 조건`: 대분류/소분류 코드와 이름
- `보유 조합/펀드`: 펀드명, 결성일, 결성총액, 투자 집행액
- `KVCA 대사`: `ASCT_ID`, 등록일, 만기일, 결성약정총액
- `TIPS 판단`: 운영사 여부, 추천 가능성, 접수 경로, 최근 추천/선정 실적
- `Why Now`: 결성일이 최근이고 투자 집행 여력이 있는 경우 우선순위 상승
- `주의`: KVCA DIVA는 벤처투자조합 중심이므로 개인투자조합, 신기사, PEF는 누락될 수 있음

### 2.6단계: TIPS 운영사 관점 확인

TIPS가 관련된 투자자는 `투자 적합도`와 `TIPS 추천 가능성`을 분리합니다. 운영사가 TIPS 운영사인지, 추천 가능 TO가 있는지, 해당 펀드 목적이 우리 회사와 맞는지, 최근 추천/선정 실적이 있는지를 확인합니다.

**검색/확인 패턴:**
```
1. "[운영사명] TIPS 운영사"
2. "[운영사명] 팁스 선정 추천 포트폴리오"
3. "[파트너명] [운영사명] TIPS LinkedIn"
4. "[운영사명] TIPS 접수처"
5. "[운영사명] [섹터] 투자"
```

LinkedIn과 개인 블로그는 공식 근거가 아니라 `업계 신호`로 분리합니다. 공개 접근 가능한 글이나 사용자가 제공한 URL만 사용하고, 비공개 페이지/개인 연락처는 저장하지 않습니다.

### 3단계: Thesis 매칭 (4차원 평가)

```
우리 스타트업 정보와 VC thesis를 비교:

차원 1: 섹터
   - VC thesis에 명시된 섹터
   - 포트폴리오 섹터 분포
   - 우리 섹터와 일치 여부
   → 🟢 MATCH / 🟡 PARTIAL / 🔴 MISMATCH

차원 2: 단계
   - VC 투자 단계 범위
   - 포트폴리오 단계 분포
   - 우리 현재 단계와 일치
   → 🟢 MATCH / 🟡 PARTIAL / 🔴 MISMATCH

차원 3: 체크사이즈
   - 전형적 투자 금액
   - 우리 조달 목표와 일치
   → 🟢 MATCH / 🟡 PARTIAL / 🔴 MISMATCH

차원 4: 지역
   - VC 지리적 초점
   - 우리 위치와 일치
   → 🟢 MATCH / 🟡 PARTIAL / 🔴 MISMATCH

종합 적합도:
   - 3개+ GREEN, RED 없음 → HIGH
   - 2개 GREEN, 또는 1개 RED → MEDIUM
   - 2개+ RED → LOW
```

### 4단계: 포트폴리오 분석

```
포트폴리오 기업을 분석하여:
1. 우리와 유사한 기업 식별 (섹터, 비즈니스 모델)
2. 섹터 분포 계산
3. 투자 패턴 파악 (Lead/Follow, 투자 빈도)
4. 경쟁사 투자 여부 체크

유사 기업 발견 시 → 웜인트로 경로 최우선
```

### 5단계: ~~data enrichment 연결 시

```
THE VC/혁신의숲/OpenDART에서 (웹 검색 기반, OpenDART는 MCP 연결 가능):
1. 투자 라운드, 밸류에이션 이력 (THE VC)
2. 포트폴리오 현황 및 투자 이력 (THE VC)
3. 스타트업 트래픽·고용·매출 성장 지표 (혁신의숲)
4. 상장사 재무제표, 지분구조 (OpenDART)
5. Exit 이력 — 상장, M&A (OpenDART + 웹 검색)
```

### 6단계: 접근 경로 매핑

```
웜인트로 경로 탐색:
1. 포트폴리오 연결
   - 유사 기업 CEO → VC 파트너
   - LinkedIn 검색, 이메일 확인

2. 기존 투자자 연결
   - 우리 투자자 → 타겟 VC
   - Co-investment 이력 확인

3. 네트워크 연결
   - LinkedIn 2촌 분석
   - 공동 배경 (학교, 회사)

~~CRM 또는 ~~knowledge base 연결 시:
   - 팀원 커넥션 자동 스캔
   - 과거 이메일 이력 검색
```

### 7단계: 출력 생성

```
리서치 리포트 생성:
1. Executive Summary
2. Thesis 적합도 (4차원 평가)
3. 펀드 프로필
4. 포트폴리오 분석
5. 최근 활동 & Why Now
6. 주요 파트너 프로필
7. 접근 경로
8. 권장 아웃리치 각도
9. 다음 단계
```

**검증 기준:**
- 한국 VC는 웹/뉴스 근거와 KVIC/KVCA 공식 근거를 섞어 쓰지 않습니다.
- KVCA에 없는 펀드는 "없음"이 아니라 `KVCA 미확인`으로 표시합니다.
- TIPS는 `투자 적합도`와 `추천 가능성`을 분리해 표시합니다.
- Seed/Pre-A 대상 리서치에는 해당 VC를 1단계 연습, 2단계 투자 가능성 검증, 3단계 리드 클로징 중 어디에 배치할지 제안합니다.

```
   - 전문성 영역

   - 주로 투자하는 섹터
   - 선호 단계

   - 컨퍼런스 발표

   - 공통 배경
   - 공유 관심사
```

## 팁

1. **포트폴리오 역추적** — 유사 기업에 투자한 VC가 가장 관련성 높음
2. **Thesis 언어 사용** — VC가 웹사이트에 쓴 언어로 아웃리치 작성
3. **Why Now 찾기** — 최근 펀드 조성, 신규 파트너, 유사 투자 = 좋은 타이밍
4. **경쟁사 투자 체크** — 동일 카테고리 재투자는 드물지만, 파트너십 각도는 가능
5. **파트너별 맞춤** — 펀드에 여러 파트너 → 가장 관련성 높은 파트너 타게팅
6. **웜인트로 최우선** — 포트폴리오 CEO 인트로가 가장 효과적
